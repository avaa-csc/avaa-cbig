Data download

Here, we distribute the global priority maps and the performance curve data of our main analyses, that together form the main results of our paper. Click the links in the table to download the data for different analysis variants:

 

Analysis variant: Data set name:
Global priorities with present land use cbig_gpan_global2000
Global priorities with future (2040) land use cbig_gpan_global2040
National priorities with present land use cbig_gpan_national2000
National priorities with future (2040) land use cbig_gpan_national2040

 

This data sets contains two main files: text file containing the data for the the performance curves and the tif file presenting a priority raster map in GeoTIFF format. Originally, the analyses were perfomed at several different resolutions: 20 km, 10 km, 5 km and to 1.7 km at the equator.  Due to the limitations in the original data, we distribute all information in 20 km resolution. Contact the authors for the result data in higher resolution.

The analysis was carried out using Zonation software developed in the Conservation Biology Informatics Group. For full methodology and references, see the metadata in datapackages and the methods section of the online paper.
